  • Produce and evaluate critically a research proposal for the chosen topic.

Jaccard Coefficient Calculations

Objective: Calculate the Jaccard Coefficient Calculations in a given table.

Learning Outcomes

  • Articulate the legal, social, ethical and professional issues faced by machine learning professionals.
  • Understand the applicability and challenges associated with different datasets for the use of machine learning algorithms.

I was given the following table and told to calculate the Jaccard Coefficient for the following pairs: "Jack and Mary", "Jack and Jim", and "Mary and Jim". To properly calculate the Jaccard coefficient, first I discarded the gender column and transformed the rest of the table into binaries.

After converting the table into binaries, we calculate the intersection between each pair. For example, Jack and Mary both have Fever and Test, so they have an intersection of 2. Jack and Jim have an intersection of 2 as well, and so does Mary and Jim. We also need to find out the union between the pairs. According to the binary table, Jack and Mary have 5 total attributes. Jack and Jim have 4, and Mary and Jim have 6. The Jaccard coefficients are as follows: Jack and Mary = 2/5 = 0.4, Jack and Jim = 2/4 = 0.5, Mary and Jim = 2/6 ≈ 0.333.

Gradient Cost Function

Objective: Run a Jupyter notebook and change the iteration number and learning_rate and observe how cost decreases.

Learning Outcomes

  • Understand the applicability and challenges associated with different datasets for the use of machine learning algorithms.

The original Gradient Cost Function had an iteration number of 100 and a learning_rate of 0.08. When running the notebook without any changes, the cost started at 89.0 at iteration 0, and ended up at cost 0.00412 at iteration 99. Changing the iteration number to be smaller resulted in no real interesting changes, as we already had records of all the iterations up to 99. However, changing the learning_rate to 0.16, double what it originally was, resulted in extremely high costs, with iteration 99 having a cost of 1.1171143851204643e+90!
